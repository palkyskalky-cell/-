# Прогнозирование эффективности лекарственных препаратов

**Автор:** И.Ю. Латыпова  
**Курсовая работа** по теме «Прогнозирование эффективности лекарственных препаратов на основе химоинформатических данных»

## Описание проекта

Цель — на основе молекулярных дескрипторов (192 признака) предсказать:
- **IC50** (ингибирующая активность),
- **CC50** (цитотоксичность),
- **SI** (индекс селективности = CC50/IC50),
а также решить бинарные задачи классификации (превышение медианы или порога 8).

## Структура репозитория

- `notebooks/` — Jupyter Notebooks с кодом:
  - `01_EDA.ipynb` — разведочный анализ данных, очистка, визуализация.
  - `02_regression_IC50.ipynb` — регрессия IC50 (XGBoost, Random Forest, CatBoost, LightGBM).
  - `03_regression_CC50.ipynb` — регрессия CC50.
  - `04_regression_SI.ipynb` — регрессия SI.
  - `05_classification_IC50.ipynb` — классификация IC50 > медианы.
  - `06_classification_CC50.ipynb` — классификация CC50 > медианы.
  - `07_classification_SI_median.ipynb` — классификация SI > медианы.
  - `08_classification_SI_8.ipynb` — классификация SI > 8.
- `data/` — исходный файл `Данные_для_курсовои_Классическое_МО.xlsx`.
- `report/` — аналитический отчёт (PDF).
- `requirements.txt` — список зависимостей.

## Результаты (кратко)

### Регрессия (лучшие модели)
| Цель | Лучшая модель | R² | MAE (ориг. шкала) |
|------|---------------|----|-------------------|
| IC50 | CatBoost | 0.447 | 181 нМ |
| CC50 | Random Forest | 0.522 | 278 нМ |
| SI   | LightGBM     | 0.272 | 33 |

### Классификация (AUC-ROC)
| Задача | Лучшая модель | AUC-ROC |
|--------|---------------|---------|
| IC50 > медианы | Random Forest | 0.826 |
| CC50 > медианы | Random Forest | 0.868 |
| SI > медианы   | Random Forest | 0.695 |
| SI > 8         | XGBoost       | 0.696 |
